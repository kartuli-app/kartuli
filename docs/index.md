## What is kartuli.app?

Kartuli.app aims to make Georgian language learning accessible and free for everyone—especially international residents, newcomers, and travelers in Georgia. 

No required accounts, no paywalls, no premium content; learning works offline and stays free.

## How to use this documentation

- Use the **sidebar** (wide screen) or **menu** (small screen) to move between sections.
- Use the **search bar** at the top to find pages by keyword.

Documentation is grouped by section:
- **Product** — Mission, vision, and how we run the project; product backlog.
- **Tech** — Development and documentation standards; technical backlog.
- **Tools** — e.g. web docs client and its CI/CD.
- **Providers** — GitHub Actions, GitHub Pages, and similar.

### For AIs

Use the [llms.txt index](https://kartuli-app.github.io/kartuli/assets/kartuli-llm.txt) to discover docs: it lists every page with a short description and URL so you can fetch only what you need. You can also browse the site like any other visitor.
