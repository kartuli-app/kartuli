# Usage examples

Phrases that should trigger this skill (no need to say the skill name):

- "Generate the PR description"
- "Draft my PR title and description"
- "What should my PR say?"
- "Write a PR description for this branch"
- "PR description for issue #123"

Optional: pass a related issue (number or link) so the description includes `Closes #N`:

- "Draft the PR description for issue #42"
- "Generate PR title and description, closes #99"
- "PR title and description for https://github.com/kartuli-app/kartuli/issues/42"
