# Kartuli

Georgian language learning platform

## Quick Start

```bash
pnpm install
pnpm dev
```

## Documentation

```bash
pnpm docs:dev     # Start documentation dev server
pnpm docs:build   # Build documentation site
pnpm docs:preview # Preview built documentation
```
